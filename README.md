# 网吧德州扑克积分赛系统

## 项目概述

专为线下网吧定制的德州扑克积分娱乐赛系统，以虚拟积分作为唯一结算依据，支持多桌独立开赛、独立计时、独立涨盲，实现网吧活动自动化管理与玩家竞技体验。

## 核心功能

### 1. 积分系统
- **仓库积分**：玩家个人账户中的积分，用于参赛带入、补分，永久留存
- **桌面积分**：玩家当前牌局中的游戏积分，淘汰清零，重新带入时重置
- **待生效补分**：已扣除仓库积分、将在下一手牌加入桌面的积分
- **总积分**：独立核算的积分体系，仅用于总积分榜排名及兑换奖励
- **等级经验值**：仅用于玩家等级提升与等级榜排名，按比赛名次增减

### 2. 比赛规则
- **桌台与人数**：单桌最低2人开局，最高9人，每桌独立计时、独立涨盲
- **初始带入**：40W-400W积分，从仓库转入桌面
- **补分规则**：发牌前可随时操作，下一手牌生效，自动校验仓库余额
- **盲注规则**：固定10分钟翻倍，可设置盲注上限
- **下注规则**：每轮操作10秒超时，无需跟注自动过牌，需要跟注自动弃牌
- **淘汰机制**：桌面积分≤0立即淘汰，进入观战状态，本手牌结束后可重新加入
- **比赛时长**：单场60分钟，支持管理员强制结束

### 3. 系统模块
- **总控管理员后台**：赛事全局配置、仓库积分管理、积分兑换管理、玩家管理、数据与排行榜管理
- **单桌管理端**：桌台配置、监控牌局、自动发牌控制、手动干预、历史查看
- **玩家客户端**：个人信息、参赛与入座、比赛桌面核心功能、补分功能、淘汰与重进、对局内历史记录功能、积分兑换入口、系统设置
- **排行榜与记录系统**：等级排行榜、总积分榜、历史冠军记录、个人战绩
- **大屏展示端**：各桌独立倒计时、当前盲注、实时排行榜、历史冠军滚动展示

### 4. 安全与合规
- **用户协议**：首次登录强制签署，明确积分无现金价值、系统故障补偿上限
- **未成年人保护**：对接网吧实名认证系统，未满18周岁禁止参与
- **防作弊机制**：同桌IP限制、异常申报、数据监控
- **操作留痕**：全流程操作日志留存，可追溯可核查

## 技术实现

- **开发语言**：Java
- **项目结构**：Maven项目
- **数据存储**：文件系统持久化，自动30秒备份，90天留存
- **并发控制**：线程安全设计，支持多桌同时运行
- **安全机制**：会话管理、IP限制、异常行为检测

## 快速开始

### 环境要求
- Java 8 或更高版本
- Maven 3.6 或更高版本

### 构建与运行
1. 克隆项目到本地
2. 进入项目目录
3. 运行 `mvn clean compile` 编译项目
4. 运行 `mvn exec:java -Dexec.mainClass="com.example.depu.TestMain"` 运行测试

### 核心类说明

#### 模型类
- `Player`：玩家信息、积分、等级等
- `Table`：桌台信息、玩家列表、游戏状态等
- `Game`：牌局信息、下注过程、胜负判定等
- `Tournament`：赛事信息、桌台管理、全局控制等
- `ExchangeItem`：兑换项目信息
- `ExchangeRequest`：兑换申请信息

#### 管理器类
- `AdminManager`：管理员操作管理
- `PlayerManager`：玩家操作管理
- `GameManager`：游戏逻辑管理
- `LeaderboardManager`：排行榜管理
- `PersistenceManager`：数据持久化管理
- `SecurityManager`：安全与合规管理

- 最大开放桌数

- 兑换项目所需总积分
- 兑换项目库存限额

## 注意事项

1. 本系统仅用于线下网吧纯娱乐积分赛，禁止用于任何形式的赌博活动
2. 积分不可转让、不可买卖、不可兑换现金，仅作为娱乐和兑换网费/饮料的凭证
3. 系统故障时，最高补偿上限为100W仓库积分
4. 连续参赛60分钟后强制休息5分钟，单日累计参赛超过4小时提示休息
